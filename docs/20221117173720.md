# Функция filter

Функция `filter` возвращает элементы коллекции подходящие под определённое условие.

![](images/filter01.svg)

Код на Python:

```python
a = [1, 2, 3, 4]
b = filter(lambda   n: n % 2 == 0, a)
print(list(b))  # [2, 4]
```
