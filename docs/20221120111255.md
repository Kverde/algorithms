# Конъюнкция

Конъюнкция — логическая операция от двух аргументов.

$$
a \land b = c
$$

[Таблица истинности](20221124151116.md) для дизъюнкции:

| a  | b | c |
|---|---|---|
| 1 | 1 | 1 |
| 1 | 0 | 0 |
| 0 | 1 | 0 |
| 0 | 0 | 0 |

В языках программирования обычно представлена операторами `and` или `&`.
